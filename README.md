FarmSmart Sierra Leone

Overview

FarmSmart Sierra Leone is a web-based Farmer Advisory System developed to support farmers by providing access to agricultural information, crop management advice, pest control guidance, and farming best practices.

The platform aims to improve agricultural productivity by making farming knowledge easily accessible through a simple and responsive website.


 Features

* Loading Screen with FarmSmart Branding
* Farmer Registration Page
* Home Page with Agricultural Information
* About Page
* Crop Advisory Page
* Pest Control Page
* Contact Form
* Responsive Design
* Interactive JavaScript Functionality


Pages

1. Loading Screen

Displays the FarmSmart Sierra Leone logo and redirects users to the registration page.

2. Registration Page

Allows farmers to enter basic information before accessing the system.

3. Home Page

Introduces the platform and highlights key services.

4. About Page

Provides information about the project's mission, vision, and objectives.

5. Crop Advisory Page

Offers guidance on:

* Rice
* Cassava
* Maize
* Groundnut

 6. Pest Control Page

Provides information on:

* Fall Armyworm
* Cassava Mosaic Disease
* Rice Blast Disease

7. Contact Page

Allows users to send inquiries and receive support.
